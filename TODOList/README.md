# Todo List на Go

Простой REST API сервис для управления задачами, реализованный на Go в рамках обучения бекенд разработке.

## Функциональность

- Создание, просмотр, обновление и удаление задач
- Хранение данных локально в памяти приложения
- REST API с JSON форматом данных
- Использование роутера Gorilla Mux

## API Эндпоинты

| Метод | URL | Описание |
|-------|-----|-----------|
| GET | `/tasks` | Получить все задачи |
| GET | `/tasks/{title}` | Получить задачу по ее заголовку |
| POST | `/tasks` | Создать новую задачу |
| PATCH | `/tasks/{title}` | Отметить выполненность/невыполненность задачи |
| DELETE | `/tasks/{title}` | Удалить задачу |

## Структура задачи

```json
{
  "title": "Заголовок задачи",
  "description": "Описание задачи",
  "complete": false,
  "createdAt": "2023-01-01T00:00:00Z"
}
```
## Запуск проекта

```sh
go run main.go
```

Сервер будет запущен на порту :8081

## Технологии

- Стандартные библиотеки Go (net/http, log, encoding/json, sync)
- Роутер Gorilla Mux
